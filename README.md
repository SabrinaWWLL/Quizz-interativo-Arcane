# 🧪 Quizz Interativo: Arcane

Um teste de personalidade e conhecimentos sobre o universo de Piltover e Zaun, baseado na aclamada série da Riot Games, *Arcane*. 

> **Status:** ✅ Concluído

## 💠 Sobre o Projeto
Este quizz foi desenvolvido para proporcionar uma experiência imersiva aos fãs da série. O projeto foca em lógica de programação para a contagem de pontos, manipulação dinâmica do DOM e um design inspirado na estética de testes de personalidade disponíveis no site da Riot Games.

## 🛠️ Tecnologias
Para este projeto, utilizei:
* **HTML5** (Semântica e acessibilidade)
* **CSS3** (Estilização temática com gradientes e sombras dinâmicas)
* **JavaScript** (Lógica de perguntas, transições de tela e cálculo de resultados)

## ⚡ Funcionalidades
- [x] Perguntas com múltiplas escolhas.
- [x] Feedback visual imediato para cada resposta.
- [x] Resultado final personalizado com base nas escolhas.
- [x] Botão para reiniciar o desafio.

## 🎨 Interface e UX
O design foi cuidadosamente planejado para remeter às duas cidades da série:
- **Tons Azuis e Dourados:** Representando a elegância dos personagens de Piltover, como Caitlyn e Viktor.
- **Tons Verdes e Roxos:** Remetendo ao caos de Zaun e seus personangens mais instáveis, como Jinx.
